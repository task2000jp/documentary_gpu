## 作品概要

| 項目 | 内容 |
|---|---|
| タイトル（案） | **勝利の福音 ── なぜ蒸気機関はスコットランドで生まれたか** |
| 旧タイトル | 祝福の系譜（v3.1） |
| 上映時間 | 約25分 |
| 章数 | プロローグ + 8章 + エピローグ |
| 言語 | 日本語ナレーション |
| 対象視聴者 | 信仰背景を問わず、歴史・文明・信仰に関心のある一般視聴者 |

**タイトル変更の理由**: 「祝福の系譜」は「繁栄の神学」に聞こえるリスクがある。
「勝利の福音」は神学の核心（Christus Victor = 恐怖と死への勝利）を前面に出し、
主論旨が「富の獲得」でなく「心の解放」であることを題名から明示する。

他の候補:
- 砕かれた鎖 ── 福音はいかにして近代文明を生んだか
- 解放の系譜 ── 心の自由が世界を動かした

## 核心メッセージ（一言）

> 産業革命がスコットランドで生まれたのは、技術的偶然ではない。
> **福音が心を解放し、その心が世界を動かしたから**。

この一文がブレたら作品として機能しない。全ての章はここに収束する。

## 論旨の骨格（5ステップ）

```
ステップ1: 神は人類に何を目指せと定めたのか？
          → 文化的使命（創1:28）＋シャローム（内外の完全な繁栄）

ステップ2: なぜ人間の心はそれに届かなくなったのか？
          → 悪魔の支配（恐怖・罪悪感・絶望）が心を縛った
            ＋ 神と人間の「断絶」が生まれた（←v3.2追加）

ステップ3: なぜ神は豊かだった文明を終わらせたのか？
          → 外は豊か・心は縛られたままの文明は自壊する（バベル、3文明）
            「福音がなかったから」は後の章で回収。ここでは伏せる（←v3.2修正）

ステップ4: 宗教改革は何を変えたのか？
          → 心の強迫ループを断ち切った。科学・医療・経済の解放へ連鎖

ステップ5: 産業革命の本当の中心に何があったのか？
          → 心の解放が先。外の解放（産業革命）はその結果。
            産業革命は宣教のインフラだった（→ IT革命・AIまで延長：v3.2追加）
```

### 第1章 — 祝福

**目的**: 神の目標（文化的使命＋シャローム）を確立する。

**主張の流れ**:
1. 創世記1:28「うめよ・ふえよ・地に満ちよ」= 命令ではなく祝福
2. シャローム = 外の豊かさ + 関係 + 健康 + 心の平和 + 神との和解（5要素）
3. ヨハネ10:10「豊かに得るために来た」
4. 整理: 神の目標は「両方」。蒸気機関はその外側の一例に過ぎない

**注意**: シャローム5要素の順次表示は、表示時間を確保すること（v3.1では後半が切れた）

**視覚的に必要なもの**:
- 創造の場面（神が人に命じる）
- 豊かに満ちた地球・文明

**利用可能画像**: `michelangelo_creation.jpg`

### 第2章 — 砕く

**目的**: 悪魔の仕組みと「勝利者キリスト」を確立する。後章の全てはここに戻る。

**主張の流れ**:
1. 目標はある。なぜ届けていないのか → 創世記3章
2. 蛇=悪魔（黙示録12:9）、霊的存在、戦場は人間の心の中
3. 悪魔の武器: 不安・罪悪感・死への恐怖（現代的に言い換え）
4. ヘブル書2:14-15「死への恐怖によって一生涯奴隷だった人々を解放」
5. 創世記3:15「おまえの頭を砕く」= 最初の福音（proto-evangelium）
6. まとめ: クリストゥス・ウィクトル = 恐怖と死への勝利

4. 神は妬んだのではなく、「心なき繁栄の加速」を止めた
5. まとめ: 外だけの豊かさが自壊するパターンの第1例

**目的**: 3文明が止まった理由を「心の縛り」として証明。バベルパターンの反復。

**主張の流れ**:

2. ギリシャ: ヘロンの蒸気装置（紀元1世紀）→ 玩具で終わった

## 神学的フレームワーク

### Christus Victor（勝利者キリスト）— 作品の主軸

グスタフ・アウレン（1931）が体系化した贖罪論の古典的類型。
十字架は「神の怒りをなだめる」(満足説)でなく、「悪魔・恐怖・死に対する勝利」(ドラマ的描写)。

この作品の論旨は富の獲得ではなく「恐怖と死からの解放＝勝利」が核心。
タイトル「勝利の福音」はここから来る。

### 文化的使命（Cultural Mandate）

創1:28「地に満ちよ・従えよ」= 被造世界の管理・開拓を召命とする。
カイパー・バーフィンクが体系化した改革派神学の柱。

産業革命は文化的使命の物質的実現として肯定できる。
**ただし**: 富は目的でなく手段・結果。使命は「シャロームの実現と福音の全伝達」。

### 一般恩恵（Common Grace）

文化的成功・科学的発見は信者/不信者を問わず神が与える恩恵。
ニュートン・ウェーバー・フロイトなど非信者の洞察を引用できる根拠。

### 繁栄の神学（Prosperity Gospel）との決定的な違い

| 論点 | 繁栄の神学 | 本作の立場 |
|---|---|---|
| 豊かさの位置 | 目的・個人の権利 | 手段・舞台装置 |
| 因果 | 信仰→個人の富（取引的） | 福音→心の解放→文明の繁栄（副産物） |
| 射程 | 個人の財布・健康 | 文明・全民族への福音 |
| 警告 | なし | 外だけの繁栄は自壊（ラオデキヤ） |

**作中での対立解消**: 第8章で明示的に切り分ける一節を入れる（E-4の台本案）。

- **厳粛・知的・映画的**。「バラエティ的演出」は排除（v3.1の反省）

### 章のつなぎ
- 暗転には必ず短いテキスト（章題または一句）を添える（v3.1の反省）

### 音響
- 全ナレーションをラウドネス正規化（例: -16 LUFS）してから合成（v3.1では章ごとに音量がバラついた）
- SEはナレーション間隙に配置。ナレと時刻衝突させない
- 「砕く」シーンのSEは低域インパクト+ブラス系（勝利の打撃として設計）
