# 二つのトラック ── 心のミクロから社会のマクロまで

本作「勝利の福音」は、**同じ原福音（悪魔に勝利したキリスト＝Christus Victor）が、
二つのスケールで働く**ことを描く。視聴者の心理型によって刺さる入口が違うので、
**トラックを分けて作り**、最後に橋で繋ぐ（多くの人は一気通貫の視点を持たないため）。

## Track A ── 内面工学（個人心理ベース・ミクロ）
- **心理型の入口**: INFJ / ISFJ 的（Fe/Ni/Si＝内面・意味・共感・ケア）。
- **出典**: `original_gospel.md`。
- **論旨**: 人は結局「恐れ」に支配されている。
  - 恐れの二大要素 = **罪悪感（過去への囚われ）** ＋ **失う恐怖（未来への不安）**。
  - これが内面ルール／ボーダー（「完璧でなければ愛されない」等）に固着し、**不幸なループ**を生む。
  - 悪魔（偽りの父）が恐れを煽り、心の配線を歪める。
- **解法**: 「**愛には恐れがない**」（Ⅰヨハネ4:18）。神は愛。恐れを無効化する素材は勇気でなく愛。
  原福音で恐れの配線を断ち、**愛のOSを入れ直す**＝内面工学の改築。
- **トーン**: 暗→温、親密、静かなアンビエント→希望。語彙は「心のシステムエンジニアリング」。
- **章**: `ch_inner_engineering`。

## Track B ── 社会・科学発展（マクロ）
- **心理型の入口**: ENTJ / INTJ 的（Te/Ni＝システム・戦略・歴史・構造）。
- **出典**: infra-payload thesis（メモリ）／自走の確定洞察（gap群）。
- **論旨鎖**: 福音 → 宗教改革（識字・啓蒙）→ 科学/医療/経済 → 産業革命 →
  **精密機械 → フィルム/録音技術 → ハリウッド/ブリティッシュロック** → IT → AI。
  芸術・大衆文化は**福音インフラ上のペイロード**。器（カメラ/レコード/IT）は価値中立、問題は何を載せるか。
- **トーン**: 荘厳・知的、歴史叙述、Knopfler風アンセム。
- **章**: `ch_why_scotland`（表題の核心）／`ch_smith_to_ai`（改革→市場→AI）／
  `ch_culture_machine`（産業→映像/音響→大衆文化）。

## 橋 ── 同じ福音、二つのスケール
- **核心洞察（ユーザー）**: **心の問題（恐れ）が解決すれば思考方向が変わり、それが探究＝科学的成長に繋がる。**
  - ミクロ: 恐れに支配された心は「生き延びること」しか考えられない。赦され・見捨てられないと知った心は、
    初めて世界へ手を伸ばす（問いを立て、測り、創る）。
  - マクロ: 宗教改革は**これを社会の規模で起こした**。地獄の恐怖・贖宥状 → 恵みによる確信 →
    確信ある自然探究 ＝ 科学。
- **設計原則**: A と B は**別章として独立**させる（型の違う視聴者に別々に届く）。
  橋の章 `ch_two_scales` だけが両者を明示的に接続し、`AdapterSpine`（背骨＝不変のペイロード）で締める。

## 受け手モデル（A/Bの下の細粒度ターゲティング）
A/B は語りの**モード**。その下に受け手の心理を二層で当てる → **`docs/psychographics.md`**。
- **エニアグラム＝恐怖/欲求のエンジン（WHAT 何を癒すか）**: 原福音が直接撃つ層。3センター
  （ハート=恥／ヘッド=不安／ガット=怒り）は `original_gospel.md` の罪悪感＋失う恐怖に
  「支配/防衛」を加えて**全範囲の恐れ**を網羅。9タイプ別に十字架で断つ嘘を定義。
- **MBTI＝認知/判断の配線（HOW どう届くか）**: 同じ真理を T(論理)/F(価値)/S(具体)/N(全体像) のどの配線に流すか。エニアグラムと直交。

## 研究システムへの含意（次の発展）

ミクロ／マクロのどちらを厚くするか `selfDeepen_` で均衡させると、全範囲カバーが自走に乗る。
