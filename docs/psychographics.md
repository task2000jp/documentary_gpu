# 受け手モデル ── 恐れ(エニアグラム) × 認知(MBTI) で全キャラクターをカバー

本作の原福音（恐れに勝利したキリスト）を、**視聴者一人ひとりの心理構成に当てて届ける**ための仕様。
`original_gospel.md`（内面工学）＝「人は結局なんらかの恐れに支配されている」を、**構造化**したもの。

## マスター変数 ＝ 上位者イメージ（親→内在化→神）
全ての恐れの源泉は「**上位者をどう捉えているか**」。幼少期の親が内在化され神イメージになる。
これが**ルート層**で、下の各層はその投影にすぎない。二つの脳で二通りに読める：
- **右脳（関係・像・情動）= 上位者との関係性そのもの**。源泉。`original_gospel.md` の「偽りの父（恐れを煽る悪魔）」が歪ませ、「神は愛」が回復させる対象。
- **左脳（理論・分類）= エニアグラム**。同じ傷を9つの恐れ/欲求として言語化した地図（＝下表）。

→ 原福音の本当のペイロードは「恐れの除去」ではなく **父イメージの回復**。これを直せば、
恐れ(エニアグラム)が解け→思考が変わり→探究に至る連鎖の全部が、上位者イメージから垂れ下がる。
この単一変数が**ミクロ(A=個人の父イメージ)とマクロ(B=宗教改革による社会の神イメージ転換)を統合**する。

## 層モデル（ルート→投影→届け方→モード）
0. **ルート: 上位者イメージ（右脳/関係）** ＝ マスター変数（上記）。
1. **エニアグラム＝恐怖/欲求のエンジン（WHAT＝何を癒すか）**: ルートの左脳的投影。原福音が直接撃つ層（下表）。
2. **MBTI＝認知・判断の配線（HOW＝どう届くか）**: 同じ真理をどの配線に流すか。
   - T→論理・因果・システムで／F→価値・関係・物語で／S→具体例・身体感覚で／N→全体像・象徴で。
   - ※エニアグラムとは直交。どのタイプもA/Bどちらの章にも入りうる（=届け方の選択肢）。
3. **語りのモード（A/B）**: Track A=内面・関係（女性的）／Track B=構造・外界（男性的）。全体トーン。→ `docs/narrative_tracks.md`

## 生成原則 ── 先鋭化（universalにしない）
**この全体像は「作り手／生成器の地図」であって、1本のコンテンツにしてはいけない。**
全層を1本に詰めると平均化して誰にも刺さらない（＝モデル崩壊）。
- **1コンテンツ＝1セル**（上位者像×エニアグラム型×MBTI配線×A/Bモード）に**先鋭化**して生成する。
- 地図は**パラメータ空間**として使う。各人は一気通貫の視点を持たない前提。
- 作り手が自然に書く ENTJ/INTJ的（構造・Track B・T配線）コンテンツも、**その層に刺さる一変種**にすぎない（無理に普遍化しない）。
- `ch_two_scales`（一気通貫）も「統合を見たがるINTJメタ層」向けの一変種。
- 既存章のセル対応例：`ch_why_scotland`/`ch_smith_to_ai`/`ch_culture_machine`＝**Track B・T・INTJ/ENTJ層**／`ch_inner_engineering`＝Track A 汎用ベース（→型別に先鋭化していく元）。

## 使い方（コンテンツ/自走への接続）
- **シナリオ**: 1テーマを「型別バリアント」で展開可能（例 `ch_inner_engineering` を型8/型4向けに差し替え）。導入フック＝その型の恐れ、解決＝上表の適用。
- **MBTIで配線**: 同じ型でも T 受けは因果図、F 受けは物語で。`graphics`(論理図)とナレーション(物語)の比率で調整。
- **自走(GAS)**: `論旨ギャップ` に `enneagram`/`center` 列を持たせ、`COGNITIVE_QUERY` に恐れ系クエリ
  （恥=shame self-worth、不安=anxiety security threat、怒り=anger autonomy control）を追加すれば、
  被造設計(ミクロ)側の研究がセンター網羅で厚くなる。現状はマクロ(Track B)偏重。

## 観測可能性 ── 生データから受け手を読む
人は社会の中で常に**上位者との関係の中で**行動するので、上位者イメージとエニアグラム傾向は
**日常の表層的な言動（コメント・投稿・話し方）に生データとして漏れ出る**。
- 怒り/支配の語（「許せない」「舐められる」「自分でやる」）→ ガット(8,9,1)。
- 恥/価値の語（「どうせ自分なんて」「認められたい」「結果が全て」）→ ハート(2,3,4)。
- 不安/安全の語（「もし〜だったら」「備えないと」「信用できない」）→ ヘッド(5,6,7)。
- 上位者像の手がかり：神/親/権威を「裁く者/見張る者/不在/気まぐれ/取引相手」のどれとして語るか。
- **システム接続**: 自走の YouTube モジュール（コメント/トレンド取得）に Groq 分類を一段足せば、
  視聴者層の**センター分布・上位者像分布**を推定 → どの型に刺さる章が不足かを `selfDeepen_` に還流。
  ＝表層言動→受け手推定→狙い撃ち を無料・自動で回せる（[[automate-dont-manual]]）。
